# 👑 Empire Game ⚔️

A real-time multiplayer party game where each player secretly submits a word, then everyone tries to figure out who submitted what.

## How It Works

1. **The host** opens the game on a laptop or TV screen
2. **Players join** on their phones by scanning a QR code — no app install needed
3. Everyone secretly submits a word (optionally within a chosen category)
4. The shuffled list of words is revealed — but not who wrote them
5. Players take turns guessing which word belongs to which person — guess right and you "capture" them into your empire
6. **Last empire standing wins!**
